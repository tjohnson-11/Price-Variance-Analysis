# Price-Variance-Analysis
MS in Data Science and Statistics project
## Project Overview
This project analyzes the price of different products to determine the impact to sales. the goal was to determine business impact of the price increases and adjust based on findings.

## Dataset
source: Kaggle
description: The dataset consisted of pricing and transaction-level data used to analyze cost variance. Key variables included expected (standard) prices, actual prices, and calculated price differences. Additional fields such as product categories, dates, and identifiers were used to segment the data and evaluate trends in price variance across different dimensions.

## Key insights
products tested were not statistically price dependent
